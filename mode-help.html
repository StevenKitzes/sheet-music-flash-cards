Modes!

Guess Mode

In Guess Mode, you will be shown a note. All you have to do is determine what that note is. After a few seconds, you'll see the name of the note so you can check if you were right. Then a new note will be shown. You can change the duration of each flash card at the bottom of the screen.

Find Mode

In Find Mode, you will be shown the full name of a note (including octave). You'll have some time to pick on the correct location of that note on the staff/ledger lines.

  • If you match the full note with octave, the screen will flash green!
  • If you match the note but not the octave, the screen will flash green, and the octave number will update to clarify which octave you picked.
  • If you pick the wrong note, the screen will flash red. You can still guess again until you're right or time runs out!
  • If you run out of time, the screen will flash red, and you'll be shown all locations of the note you're looking for.
